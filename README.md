# SAP-ADLS-IncrementalLoad
<img width="1629" height="627" alt="image" src="https://github.com/user-attachments/assets/66e50d63-9455-4faa-a5f9-6c2cc9656b58" />
This project demonstrates an end-to-end data engineering pipeline built using Azure Data Factory (ADF). The pipeline extracts data from SAP HANA, processes it across multiple layers, and stores it in Azure Data Lake Storage (ADLS) and SQL Server using a Medallion Architecture (Source → Silver → Gold).
